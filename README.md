# Tab Notes

A lightweight Chromium extension for taking quick notes from the browser popup. Notes are automatically saved using browser local storage.

## Features

- Quick note-taking from the toolbar popup
- Automatic saving while typing
- Persistent storage using browser local storage
- Simple and clean interface
- Lightweight and fast

## Technologies Used

- HTML
- CSS
- JavaScript
- Chrome Extension Manifest V3
- Browser Local Storage
